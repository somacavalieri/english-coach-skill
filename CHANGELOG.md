# Changelog

All notable changes to the English Coach skill will be documented in this file.

## [1.0.2] - 2026-04-13

### Added
- Initial versioning setup with semantic versioning
- Clarified rule: "Do NOT correct punctuation, capitalization, accents, or other minor typographical details"
- Only correct grammar errors, wrong word choices, unnatural phrasing, and subject-verb/noun-adjective agreement issues

### Fixed
- Synced skill definitions between SKILL.md and english-coach.skill

## [1.0.0] - Initial Release

### Features
- Silent English writing coach for grammar and phrasing feedback
- Non-intrusive coaching notes in conversation
- Logging of corrections to markdown file
- Clear before/after format with strikethrough and bold styling
